# Сводные Таблицы И Поиск

Кариотип должен давать не только картинки, но и сводные данные. Таблицы помогают быстро понять, какие образцы уже обработаны, сколько есть метафаз, какие зонды использовались и какие аномалии найдены.

## Зачем Нужны Таблицы

Таблицы закрывают рабочие вопросы:

- какие образцы уже имеют результат;
- сколько растений и препаратов связано с образцом;
- сколько метафаз импортировано;
- какие зонды использованы;
- сколько хромосом извлечено;
- где есть аномалии;
- какие данные еще не размечены;
- что можно экспортировать.

Это не замена атласу, а рабочий обзор состояния данных.

## Основная Сводка По Образцам

Строка образца может включать:

- ID образца;
- вид или линия;
- родители и поколение;
- число растений;
- число препаратов;
- число окрашенных препаратов;
- число метафаз;
- число извлеченных хромосом;
- использованные зонды;
- наличие лицевого кариотипа;
- число экспортов;
- список аномалий;
- статус.

Такая таблица помогает увидеть, где работа уже закончена, а где данные зависли.

## Сводка По Метафазам

Для метафаз полезны:

- образец;
- препарат;
- окрашенный препарат;
- PSD или фото;
- координаты;
- количество слоев;
- количество хромосом;
- количество размеченных хромосом;
- количество идеограмм;
- качество;
- статус.

Эта таблица нужна для контроля импорта и выбора лучших метафаз.

## Сводка По Зондам

Для зондов важно видеть:

- какие панели использовались;
- в каком канале был каждый зонд;
- к каким образцам и гибридизациям они относятся;
- какие классы хромосом размечены по этим зондам;
- где есть аномальные сигналы.

Эта информация будет особенно полезна для будущего атласа.

## Сводка По Аномалиям

Таблица аномалий должна показывать:

- образец;
- тип аномалии;
- субгеном и класс;
- конкретные хромосомы;
- уровень уверенности;
- подтверждение по метафазам;
- комментарий;
- есть ли подпись в экспортном обзоре;
- ссылка на кариотип или идеограмму.

Сводка помогает искать повторяющиеся случаи и готовить материал для обсуждения.

## Поиск

Минимальный поиск должен работать по:

- ID образца;
- текстовому имени образца;
- зондам;
- субгеному;
- классу хромосомы;
- типу аномалии;
- статусу;
- наличию идеограммы;
- наличию экспорта.

Примеры запросов:

- все образцы с `5D`;
- все случаи `трисомии`;
- все кариотипы с `GAA+pAs1`;
- все хромосомы без идеограммы;
- все результаты по образцу `1730.25`.

## Граница С Атласом

Сводные таблицы кариотипа отвечают на вопрос `что уже обработано и где лежит результат`.

Атлас отвечает на вопрос `что эти результаты значат в сравнении с другими образцами, зондами и литературой`.

Если пользователь из таблицы аномалий хочет сравнить похожие случаи, это уже переход в атлас.
