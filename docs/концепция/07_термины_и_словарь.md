# Термины И Словарь

Единый словарь Karyolab v2. Если в любом другом документе термин используется по-другому, документ нужно поправить, а не словарь.

## Лабораторные Объекты

| Термин | Что это |
|---|---|
| Образец | Якорь системы. Физический материал (семена одного источника, гибрид, линия). Создаётся вручную в журнале. |
| Растение | Один проросток из образца. |
| Смесь растений | Несколько проростков из одного образца, использованных вместе (для одного препарата). |
| Препарат | Физическое стекло с давленным материалом. Делается из растения или смеси. |
| Окрашенный препарат | Препарат после конкретной гибридизации с конкретным набором зондов. У одного препарата может быть несколько окрашенных препаратов (после переотмывки). |
| Фотография метафазы | Сырая фотография под микроскопом, прикреплённая к окрашенному препарату. |
| Метафаза | Выбранный участок фотографии, на котором видна полноценная разложенная картинка хромосом. |
| Хромосома | Отдельный объект, извлечённый из метафазы (один слой PSD). |
| Идеограмма | Схема хромосомы с центромерой, сигналами и аномалиями. |

## Кариотипы

| Термин | Что это |
|---|---|
| Кариотип метафазы | Кариотип, собранный по одной метафазе. ID `<метафаза>.kar.<n>`. |
| Кариотип образца | Кариотип, собранный из лучших хромосом всех метафаз гибридизации. ID `<образец>.kar.<n>`. У образца может быть несколько кариотипов. |
| Эталонный кариотип / Теоретический кариотип | **Синонимы.** Кариотип образца с пометкой "эталонный/теоретический" или чисто теоретическая запись из литературы. |
| Сохранённое сравнение | Объект атласа, фиксирующий собранную раскладку (образцы, эталоны, режим). |
| Экспорт изображения | Результирующий файл (TIFF) для публикации. Хранится у образца. |

## Ивенты

| Термин | Что это |
|---|---|
| Засев / проращивание | Ивент начала жизни образца. Длинный таймлайн. |
| Создание препарата | Ивент создания препарата от растения или смеси. |
| Отмывка препаратов | Общий ивент на партию препаратов. Имя по умолчанию `WP-{YYYY-MM-DD}`. |
| Гибридизация | Ивент окраски препарата зондами. Создаёт окрашенный препарат. |
| Фотографирование | Ивент съёмки препарата под микроскопом. Решает судьбу препарата. |
| Свободный ивент / заметка | Текстовая запись без изменения статусов. Заметка не показывается в календаре, только в ленте дня. |

## Статусы

| Термин | Что это |
|---|---|
| `создан` (препарат) | Препарат сделан, ничего ещё не происходило. |
| `отмыт` (препарат) | Препарат прошёл отмывку, готов к гибридизации. |
| `гибридизован` (препарат) | Есть окрашенный препарат. |
| `сфотографирован, судьба не решена` (препарат) | После фото оператор может решить позже. |
| `переотмыт` (препарат) | Препарат отмыт повторно для новой гибридизации. |
| `выброшен` (препарат) | Конец жизни. |
| `решу позже` (фото-судьба) | Оператор отложил решение. Решение принимается из карточки препарата. |
| `черновик` (кариотип образца) | Оператор работает над кариотипом. |
| `утверждён` (кариотип образца) | Оператор подтвердил, можно экспортировать. |
| `экспортирован` (кариотип образца) | Есть хотя бы один экспорт. |
| `архивный` (кариотип образца) | Устаревший, скрыт из активной работы. |

## Хромосома И Идеограмма

| Термин | Что это |
|---|---|
| Канонический ID хромосомы | `<метафаза>.cNN`. Присваивается сразу при импорте PSD. **Не меняется**. |
| Имя файла хромосомы | До разметки: `untitled-N`. После разметки: `..._5D.png` (с классом+субгеномом). |
| Класс+субгеном | Объединённое понятие. Запись `5D` означает класс `5` в субгеноме `D`. Раздельных справочников нет. |
| Шкала размера и центромеры | Одна шкала, **три ползунка**: верхняя точка размера, нижняя точка размера, центромера. |
| Тип сигнала | Один из четырёх: `маленькая точка`, `точка`, `большая точка`, `отрезок`. На идеограмме сигнал — это две точки рядом на одном уровне. |

## Аномалии

| Термин | Что это |
|---|---|
| Хромосомная мутация | Аномалия на одной хромосоме (привязка к идеограмме). Тип берётся из справочника атласа. |
| Геномная мутация | Аномалия на уровне кариотипа образца. Тип берётся из справочника атласа. |
| Замещение, трисомия, моносомия, нуллисомия | Это **типы геномных мутаций**, не предзаготовленные бейджи в UI. Названия и описания редактируются оператором. |
| Незаполненность как аномалия | Сознательное отсутствие класса, отмеченное как геномная мутация. Позволяет экспортировать неполный кариотип. |

## Атлас И Справочники

| Термин | Что это |
|---|---|
| Эталон / теоретический кариотип | Синонимы. См. выше. |
| Сохранённое сравнение | Объект атласа. См. выше. |
| Шаблон субгеномов | Готовый набор субгеномов для вида (пшеница, тритикале, рожь, эгилопс) или пользовательский. Редактируется оператором. |
| DAPI | Флюорохром синего канала. Всегда автоматически синий, оператор не выбирает канал для DAPI. |
| Тильт | Индикатор раздражения оператора. Один уровень (нет градаций). Считается годовой показатель `тильт / результат`. |

## UI

| Термин | Что это |
|---|---|
| Главное (боковое) меню | Слева. Содержит верхние пункты: `Журнал`, `Кариотип`, `Атлас`. |
| Горизонтальные вкладки | Сверху страницы внутри раздела. В кариотипе: `Импорт`, `Кариотип`, `Экспорт`. В атласе: `Матрица`, `Сравнение`, `Эталоны`, `Сохранённые сравнения`, `Справочники`. **Не дублируются** в боковом меню. |
| Ивент-стак | Один значок в календаре, объединяющий несколько ивентов одного типа в один день. Клик ведёт на отдельную страницу со всеми карточками этого дня (не поповер). |
| Полупрозрачный блок | Способ отображения длинного периода в календаре (например, "перенос в холодильник → 2 недели до созревания"). |

## Языковые Правила

- Все UI-элементы и подписи — на **русском**.
- Палитра — **болотно-зелёная** (см. дизайн-доки).
- Никаких медицинских терминов (`patient`, `case`, `clinical summary`).
