## Общие Дизайн-Правила

Эти правила применяются во всех разделах:

### Язык И Палитра

- Все UI-элементы и подписи — на **русском**.
- Палитра — **болотно-зелёная** (см. [журнал/12_дизайн_журнала.md](../../журнал/12_дизайн_журнала.md)).
- Никаких медицинских терминов (`patient`, `case`, `clinical summary`).

### Навигация

- Боковое меню содержит верхние пункты: `Журнал`, `Кариотип`, `Атлас`.
- Подразделы (импорт/кариотип/экспорт в кариотипе; матрица/сравнение/эталоны/справочники в атласе) — **горизонтальные вкладки сверху**, не пункты бокового меню. Это сделано, чтобы освободить рабочее пространство.

### Календарь И Лента

- Год выбирается отдельно (нет кнопки "сегодня").
- Несколько ивентов одного типа в один день агрегируются в один значок-стак. Клик ведёт на **отдельную страницу со всеми карточками**, не поповер.
- Длинный период (например, "перенос в холодильник → 2 недели до созревания") отображается полупрозрачным блоком на календаре.
- Заметки **не показываются в календаре**, только в ленте дня.

### Прогресс

- Колонки: `созрели`, `ждёт отмывку`, `отмыт`, `гибридизован`, `есть результат`.
- Все колонки, кроме `есть результат`, показываются полными списками.

### Карточки Образца

- Кариотипы образца показываются как **набор плиток с превью** (несколько штук может быть).
- Кариотипы метафаз — отдельный список.
- Сохранённые сравнения из атласа — отдельный набор плиток.
- Карточка отмывки **не дублирует** список образцов партии — он есть в таблице ивента в виде кликабельных ссылок.

### Разметка Идеограммы

- Шкалы рядом с изображением, **в одинаковом масштабе по высоте**.
- Шкала размера и центромеры — одна, **три ползунка** (две точки размера + центромера).
- Четыре типа сигналов: маленькая точка, точка, большая точка, отрезок. На идеограмме сигнал — **две точки рядом**.
- Аномалия добавляется через поиск по справочнику атласа + кнопка "добавить новый тип".

### Разметка Генома

- Кнопка автораскладки называется **`Разложить по классам`** (не "auto-sort").
- Программа только предлагает, не решает. Особенно для аномалий.
- В UI **нет предзаготовленных бейджей** "трисомия / моносомия / нуллисомия / замещение" — всё через справочник геномных мутаций.

### Экспорт

- Изображения — **TIFF** (максимальное качество, без ползунка).
- Таблицы — **Excel/текст**, не картинка таблицы.
- Подписи аномалий — со стрелками, не перекрывают хромосому или идеограмму.
- Цвета каналов: текст подписи зонда красным/зелёным/синим.

### Сообщения Об Ошибках

- Понятные русские формулировки.
- Рядом с действием, а не в общем баннере.
- Не блокируют работу: показывают, что не так, но позволяют сохранить черновик.
