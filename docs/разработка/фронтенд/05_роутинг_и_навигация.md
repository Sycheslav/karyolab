# Роутинг И Навигация

## Структура Маршрутов

```
/                                       → редирект на /журнал
/журнал                                  → главная журнала (календарь + прогресс)
/журнал/образцы                          → список всех образцов
/журнал/образцы/:sampleId                → карточка образца
/журнал/ивенты/:eventId                  → карточка ивента
/журнал/день/:date                       → лента дня (все ивенты дня)
/журнал/прогресс                         → доска прогресса
/журнал/заметки                          → заметки

/кариотип                                → редирект на /кариотип/импорт
/кариотип/импорт                         → вкладка импорта
/кариотип/импорт?okrPrep=:id             → импорт с предзаполненным окрашенным препаратом
/кариотип/разметка                       → вкладка разметки (по умолчанию хромосома)
/кариотип/разметка/хромосома/:chromId    → разметка идеограммы
/кариотип/разметка/геном                 → разметка генома (метафаза)
/кариотип/разметка/геном?level=hyb       → разметка генома (гибридизация)
/кариотип/экспорт                        → вкладка экспорта

/атлас                                   → редирект на /атлас/матрица
/атлас/матрица                           → матрица класс × субгеном × образец
/атлас/сравнение                         → два кариотипа рядом / мультивыбор
/атлас/эталоны                           → список эталонных/теоретических кариотипов
/атлас/сохранённые                       → сохранённые сравнения
/атлас/справочники                       → редирект на первый справочник
/атлас/справочники/виды
/атлас/справочники/зонды
/атлас/справочники/классы
/атлас/справочники/хромосомные-мутации
/атлас/справочники/геномные-мутации
/атлас/справочники/теоретические-записи
```

## Боковое Меню

Содержит ровно три верхних пункта:

- `Журнал`
- `Кариотип`
- `Атлас`

Внизу — две дополнительные кнопки:

- `+` — быстрое создание (модалка с выбором: образец, ивент, заметка).
- `ВСЕ` — общий список всех объектов (поиск/фильтр).

Подразделы (импорт/кариотип/экспорт; матрица/сравнение/...) **не дублируются** в боковом меню. Они только в горизонтальных вкладках сверху страницы.

## Горизонтальные Вкладки

Кариотип:

- `Импорт` (`/кариотип/импорт`)
- `Кариотип` (`/кариотип/разметка`)
- `Экспорт` (`/кариотип/экспорт`)

Атлас:

- `Матрица`
- `Сравнение`
- `Эталоны`
- `Сохранённые сравнения`
- `Справочники`

Активная вкладка подсвечена акцентным цветом.

## Глубокие Ссылки

Из карточек и ивентов можно сразу попасть в нужное место кариотипа:

- из карточки фотографирования → `/кариотип/импорт?okrPrep=<id>`;
- из карточки хромосомы → `/кариотип/разметка/хромосома/<chromId>`;
- из карточки кариотипа образца → `/кариотип/экспорт?source=<karyotypeId>`.

При переходе фронт восстанавливает контекст: предзаполняет селекты, открывает нужные панели, не заставляет искать заново.

## Возврат К Источнику

Из любой ячейки атласа должен быть доступен переход к источнику данных:

- к идеограмме → `/кариотип/разметка/хромосома/<chromId>` (открывается в режиме просмотра, не редактирования, если хромосома уже частью утверждённого кариотипа);
- к карточке образца → `/журнал/образцы/<sampleId>`;
- к карточке ивента → `/журнал/ивенты/<eventId>`.
