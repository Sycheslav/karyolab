# Локализация И Дизайн

## Язык

Karyolab v2 — **только на русском** в UI. Никаких медицинских терминов (`patient`, `case`, `clinical summary`, `specimen`).

Допустимые исключения:

- технические термины в коде (имена переменных, типов);
- имена файлов в системе (могут быть на латинице из-за совместимости).

В UI ничего из латиницы:

- ❌ `Save as draft` → ✅ `Сохранить черновик`
- ❌ `Print Label` → ✅ удалено (раньше была кнопка, теперь не нужна)
- ❌ `Core Identification` → ✅ `Идентификация`
- ❌ `Auto-sort` → ✅ `Разложить по классам`
- ❌ `Commit 8 Files` → ✅ `Сохранить 8 хромосом`

## Палитра

Болотно-зелёная палитра. Точные значения цветов задаются в `tailwind.config.js`. Базовые роли:

| Роль | Цвет |
|---|---|
| Основной фон | светлый болотно-зелёный |
| Тёмный холст (разметка идеограммы, матрица генома) | глубокий зелёно-чёрный |
| Акцент / активные элементы | мшисто-зелёный |
| Текст основной | тёмный графитовый |
| Текст вторичный | приглушённый серо-зелёный |
| Красный канал зонда | насыщенный красный (используется только для подписей) |
| Зелёный канал зонда | насыщенный зелёный (только для подписей) |
| Синий канал DAPI | насыщенный синий (только для подписей) |
| Предупреждение | тёплый янтарный |
| Ошибка | приглушённый красный (не похожий на красный канал) |

Подробности — в [журнал/12_дизайн_журнала.md](../../журнал/12_дизайн_журнала.md).

## Типографика

- Основной шрифт — без засечек, читаемый при мелком кегле (Inter, Roboto или системный).
- Размеры строго по шкале Tailwind: `text-sm`, `text-base`, `text-lg`, `text-xl`, `text-2xl`.
- Заголовки экранов — `text-2xl font-semibold`.
- Заголовки разделов карточки — `text-base font-semibold`.
- Подписи в формах — `text-sm`.
- ID объектов и пути — `font-mono text-sm`.

## Компоненты

### Боковое Меню

Содержит ровно три верхних пункта: `Журнал`, `Кариотип`, `Атлас`. Внизу — кнопки `+` (быстрое создание) и `ВСЕ` (общий список объектов).

Подразделы (Импорт/Кариотип/Экспорт; Матрица/Сравнение/...) **не появляются** в боковом меню.

### Горизонтальные Вкладки

Сверху страницы внутри раздела. Активная вкладка выделена акцентным цветом и нижней чертой.

### Календарь

- Селектор месяца + селектор года (отдельный).
- Нет кнопки "сегодня".
- Ивенты одного типа в один день — один значок-стак с числом.
- Длинные периоды — полупрозрачный блок поверх дней.

### Карточки

- Образец, ивент, кариотип, метафаза.
- Карточка кариотипа на карточке образца — плитка с превью изображения и подписью.

### Формы

- Все надписи на русском.
- Кнопка сохранения чётко описывает действие: `Сохранить черновик`, `Сохранить ивент`, `Сохранить идеограмму`, `Сохранить 8 хромосом`.
- Ошибки показываются рядом с полем, а не общим баннером сверху.

## Доступность И Размеры

- Минимальная высота кликабельных элементов — 32px (компактный UI приоритет).
- Контраст текста к фону — не ниже WCAG AA.
- На рабочих экранах кариотипа левое меню можно сворачивать, чтобы освободить место для холста.
