## Когда «есть результат»

_тема: Согласованность статусов_

> «В разделе кариотипа создан минимум один кариотип» (журнал) vs «утверждённый результат становится виден атласу» (атлас).

### Q2.1 — Когда образец должен переходить в статус «есть результат»?

**Ответ:** Только когда из него собран экспортный обзор (картинка для статьи)

**Уточнение:** понятие лицевой кариотип - относится только к собранным из нескольких метафаз кариотипам. если кариотип сформирован из одной метафазы - метафазный кариотип. результат на уровне софта и моей работы - сформированный кариотип образца. я могу его назначить таковым и тогда появляется статус есть результат. экспорт ситуативен, но в целом, почему бы и не довести это до экспорта, если готовый кариотип уже есть, то пусть просто будет храниться файл.  может быть больше одного сборного кариотипа на образец (с разных растений, например)

### Q2.2 — Можно ли «снять утверждение» с кариотипа, если потом нашлась ошибка?

**Ответ:** Да спокойно — статус откатывается, образец возвращается «в работе»

### Q2.3 — Какие статусы реально нужны лицевому кариотипу?

**Уточнение:** в работе (ил черновик) - является текущим лицевым/сборным кариотипом (то есть экспортирован, продолжая логику прошлых вопросов)
